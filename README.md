# AtlasGo — UX Case Study

A location-based travel planning app for solo 
adventurers, families, and eco-conscious tourists.

## Overview
- Role: Sole UX/UI Designer
- Duration: 8 Weeks
- Tools: Figma, FigJam, Whimsical, Google Forms

## Problem
Travelers struggle to discover authentic local 
experiences beyond mainstream tourist spots.

## Solution
AtlasGo provides personalised itinerary planning,
local business discovery, and offline navigation
in one seamless mobile experience.

## Key Research Findings
- 65% of users were solo travelers
- 76% wanted local business visibility  
- 79% needed reduced tourist overcrowding
- 85% believed personalised planning would 
  improve their experience

## Deliverables
- User Research & Personas
- User Journey Map (9 stages)
- Information Architecture
- Lofi & Hifi Wireframes
- Design System
- Interactive Prototype
